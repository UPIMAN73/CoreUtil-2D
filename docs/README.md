## Getting Started
To get started with CoreGameUtility, you need to follow these steps:
1. Install the CoreGameUtility plugin for your game engin
2. Import the CoreGameUtility script into your project.
3. Configure the plugin settings in the plugin configuration file (plugin.cfg).
4. Follow the instructions provided by the CoreGameUtility plugin in order to properly configure and use the plugin
